# Welcome to the Documentation

This is the home page of your documentation site.

## Features

- **Markdown Support**: Write your docs in markdown
- **Beautiful Theme**: Material-inspired design with Pico CSS
- **Fast & Lightweight**: Built with Go
- **Easy to Deploy**: Single executable file

## Getting Started

1. Add your markdown files to the `docs/` directory
2. Run the server
3. Access your documentation at `http://localhost:3005`

## Code Example

package main

import "fmt"

    fmt.Println("Hello, Documentation!")
}
